# Multi-Booting

## What is Multi-Booting?

Multi-booting is the process of installing multiple operating systems on a single computer and choosing which one to start during boot.

My current setup consists of:

- Windows 11
- Arch Linux

This allows me to use both operating systems for different purposes while gaining hands-on experience with system administration and troubleshooting.
